# Sistema de Gerenciamento de Projetos

Projeto desenvolvido em Java utilizando Programação Orientada a Objetos.



##  Funcionalidades

- Cadastro e login de usuários
- Perfis de acesso (Administrador, Gerente e Colaborador)
- Cadastro, listagem e remoção de usuários
- Cadastro e gerenciamento de projetos
- Alteração de status de projetos
- Cadastro e gerenciamento de equipes
- Associação de equipes a projetos
- Validações de dados (CPF, e-mail e login)



## Persistência de Dados

O sistema utiliza arquivos `.txt` para armazenamento de usuários, garantindo persistência simples de dados.



## Estrutura do Projeto

- `model` → Classes principais (User, Project, Equipe)
- `service` → Regras de negócio
- `repository` → Persistência em arquivos TXT



## Tecnologias

- Java
- Programação Orientada a Objetos
- Git e GitHub



##  Execução

Executar a classe `Main` no IntelliJ IDEA ou outra IDE Java.
